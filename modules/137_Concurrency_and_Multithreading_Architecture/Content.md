**Recommended level:** 
=======================
- Junior / Middle

**Topics:**
=======================
- Java memory model
- Concurrency vs parallelism vs multithreading
- Thread API
- Synchronization & Locks
- Volatile, Atomics & AtomicReference
- Thread pools & executors
- Other concurrent package possibilities 

**Required time for the course:** 
=======================
- Lectures: ~10h
- Tasks: ~15h mandatory
