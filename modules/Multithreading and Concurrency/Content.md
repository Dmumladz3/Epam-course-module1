**Recommended level:** 
=======================
- Junior / Middle

**Topics:**
=======================
- What is concurrency
- Thread API
- Java memory model
- Semaphore, CountDownLatch, CyclicBarrier
- Completable future, parallel streams 
- Anatomy of Synchronization
- Locks
- Atomics
- Thread pools
- Concurrent collections

=======================
- Lectures: ~10h
- Tasks: ~15h mandatory
